# Lab Audio & Video – Unity (Chương 4)

Project thực hành môn học Unity – Chương 4: Audio & Video.  
Bao gồm các Lab từ 1 đến 7 + Mini Project Intro Cutscene Game.

## Mục tiêu project
- Nắm vững hệ thống Audio (AudioSource, AudioListener, AudioClip, Spatial Audio 2D/3D).  
- Sử dụng VideoPlayer, RenderTexture, UI RawImage, Material Override.  
- Xử lý event (prepareCompleted, loopPointReached).  
- Tối ưu AudioClip (Load Type, Compression Format).  
- Xây dựng màn intro cutscene với video + BGM + nút Skip + chuyển scene.
